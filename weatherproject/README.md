# Django Weather App

**Technologies:** Python, Django, HTML, CSS, OpenWeather API, Git, GitHub

## Project Overview
This is a web-based weather application built with Django. Users can search for any city and get real-time weather information including temperature, humidity, and weather conditions. The data is fetched dynamically using the OpenWeather API.

## Key Features
- Search for weather by city
- Displays temperature, humidity, and weather conditions
- Responsive UI design using HTML & CSS
- Organized Django project structure for easy maintenance
- API integration for live data

## Skills Demonstrated
- Backend development with Django
- API integration
- Template rendering and frontend display
- Git version control and GitHub repository management
